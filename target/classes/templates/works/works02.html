～事業所での成果物について～

インターン活動にて作成を行いました、母子健康手帳のWebアプリケーションになります。

ログイン機能をはじめ、CRUD機能（データの閲覧、登録、更新ができる）、権限に応じて可能な操作の制限、を本アプリケーションに組み込みました。

権限について、
「利用者（母子健康手帳をもつ母親）」と
「支援者（産婦人科など病院に勤務する医師の方）」で分けております。

利用者の方と支援者の方でデザインUIをそれぞれ変更して対応しました。 利用者の方は、PCよりもスマホでの閲覧機会が多いためスマホ推奨の見やすいシンプルなデザインとしました。 一方で支援者の方は、各利用者ごとにすばやくデータを記入閲覧出来た方がよいためPC推奨で、なるべく一画面に必要情報がまとまるように工夫しました。

今回のアプリは以下技術を使用して作成

言語 Java
DB PostgreSQL
フレームワーク SpringBoot
